# [react-native-tutorial](https://g6ling.gitbooks.io/react-native-tutorial-korean/content/)
## How to Start ?
```bash
$ npm install -g create-react-native-app
$ create-react-native-app AwesomeProject
$ cd AwesomeProject
$ npm start
```
